# Next.js Template Project

## 启动方式

```bash
# 开发环境启动（使用 turbopack，端口 18888）
pnpm dev

# 构建项目
pnpm build

# 生产环境启动
pnpm start

# 代码检查
pnpm lint
```

## 技术栈

- **框架**: Next.js (15.4.5)
- **UI 渲染**: React (19.1.0)
- **UI 组件**:
  - Radix UI
  - react-icons
- **样式**: Tailwind CSS (v4)
- **状态管理**: Zustand
- **网络请求**: Axios
- **工具库**:
  - dayjs (日期处理)
  - lodash-es (工具函数)
  - nprogress (加载进度条)
  - react-hot-toast (通知提示)
- **开发工具**: TypeScript, ESLint
